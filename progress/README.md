# Project Progress

本目录用于记录 Annie TaskGraphScheduler 的项目进度。

## 目录约定

```txt
progress/
└── phase-XX/
    ├── phase-summary.md
    └── tasks/
        ├── task-001-xxx.md
        └── task-002-xxx.md
```

## 状态约定

| 状态 | 含义 |
|---|---|
| todo | 尚未开始 |
| in_progress | 正在进行 |
| blocked | 被依赖、决策或环境阻塞 |
| review | 已完成实现，等待审查或验证 |
| done | 已完成并验证 |
